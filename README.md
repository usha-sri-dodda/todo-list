# To-Do List Application

A full-stack task management application that allows users to create,
update, and delete tasks.

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MySQL

## Features

- Create tasks
- Update tasks
- Delete tasks
- Persistent task storage

## Run Locally

git clone https://github.com/usha-sri-dodda/todo-list
cd todo-list
npm install
npm start
